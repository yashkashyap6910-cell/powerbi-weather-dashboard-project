# 🌦️ Power BI Weather Dashboard

## 📊 Overview
This project is a Power BI dashboard that provides insights into weather conditions and air quality. It helps users analyze environmental data through interactive and visually appealing charts.

---

## 🚀 Features
- 🌡️ Temperature trends and forecast
- 💧 Humidity and wind speed analysis
- 🌅 Sunrise and sunset timings
- 🌫️ Air Quality Index (AQI) visualization
- 🌧️ Chance of rain prediction
- 📍 Multi-city comparison
